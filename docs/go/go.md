# go

## 特性

- 大写字母开头的函数才能被外部包调用，小写字母开头的函数只能在包内使用
- `if err != nil` 是 Go 语言的惯用法，错误通常是返回值的最后一个，例如`result, err := func()`
- [[goroutine]]
- 反射：在运行时更新变量和检查它们的值、调用它们的方法，而不需要在编译时就知道这些变量的具体类型
