# 模式匹配

[[rust]] 版。python版见[[python#match]]

## `match` 匹配

`match` 的匹配必须穷尽所有情况，`_` 表示其他情况。

```rust
match target {
    模式1 => 表达式1,
    模式2 => {
        语句1;
        语句2;
        表达式2
    },
    _ => 表达式3
}
```

## `if let`

只要匹配一个条件，且忽略其他条件时就用 `if let` ，否则都用 `match`。

```rust
if let Some(3) = v {
    println!("three");
}
```

## `matches!`宏

将一个表达式跟模式进行匹配，然后返回匹配的结果 `true` or `false`。

```rust
let foo = 'f';
assert!(matches!(foo, 'A'..='Z' | 'a'..='z'));
```
