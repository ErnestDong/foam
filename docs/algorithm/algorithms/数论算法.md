# 数论算法

## 欧几里得算法

取最大公约数有 $\operatorname{gcd}(a,b)=\operatorname{gcd}(b, a \mod b)$

取模运算可以用[[群论]]结构描述，取模意义上的加法/乘法等式是一个有限交换群。

## 寻找大素数

素数分布函数$\pi(n)$为小于等于 n 的质数的数目，满足$\lim_{n \to \infty} \frac{\pi(n)}{n / \ln n}=1$
