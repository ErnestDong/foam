# DNS

DNS 协议是基于 [[UDP]] 的，也是其他协议如[[http]]、[[smtp]]、ftp常用的服务，是一个分布式[[数据库]]作用有

- 将主机名翻译为 [[IP]]地址
- 主机/邮件服务器别名
- 负载分配

## DNS 的实现

根 DNS 服务器 -> 顶级域 DNS 服务器 -> 权威 DNS 服务器 的层级实现，此外还有本地 DNS 服务器记录局域网的 IP。
查询是递归的，为了降低时延，可以采用 DNS 缓存
