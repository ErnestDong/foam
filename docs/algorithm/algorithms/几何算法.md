# 几何算法

几何算法要尽量避免除法运算带来的精度损失

## sweeping

求一组线段是否存在交点。假定没有垂直于 x 轴的线段，我们在每一个端点处作垂线垂直于 x 轴，按照交点的 y 值排序，如果不存在相交，那么每个垂线的交点都是保序的，例如与线段 a 交点永远高于线段 b 交点

## 凸包

定义点集 Q 上 $P=CH(Q)$，P 是包含了 Q 所有点的最小凸多边形，凸包顶点一定属于 Q，计算方式常用旋转 sweeping 方式
