# 线程

线程是[[os]]调度资源的最小单位。

在任何一个时间点上，线程是 joinable 或 detachable。`join` 阻塞当前线程，直到某个线程结束；`detach`则相反，使得某个线程分离，即不可被一个线程回收

## 线程的共享变量

每个线程有自己的 context，包括线程 ID、栈和栈指针、程序计数器、条件码和通用目的寄存器。而其他的[[进程]] context 是共享的，如[[虚拟内存]]中的只读文本、全局变量、读写数据、堆
