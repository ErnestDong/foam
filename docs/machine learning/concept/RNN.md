# RNN

![RNN](../../attachments/rnn.png)

很多被 [[transformer]] 取代，权值共享，有“记忆”功能。

- 难以并行，计算性能差
- 过去的隐藏状态可能会被丢掉

## LSTM

- 忘记门 F：将值朝 0 减小
- 输入门 I：决定是否忽略输入数据
- 输出门 O：决定是否用隐状态

cell 的状态不仅有隐藏层 h，还有一个 $C_t=F_t\bigodot C_{t-1}+I_t\bigodot \tilde{C}_t$ ，$\tilde{C}_t$ 是根据 输入和 H 算出来的
h 由 C 和 O 决定
