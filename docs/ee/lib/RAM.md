# 内存

## DRAM

- 由一个电容+一个[[cmos]]构成，电容存储 1 或 0，CMOS 用来控制读取和写入
- 电容可能会漏电，所以需要不断刷新

![电路图](../../../attachments/dram.png)

## SRAM

- 由一个[[时序逻辑电路#RS 锁存器]]和与门构成，RS 锁存器存储 1 或 0
- 不需要刷新

![电路图](../../../attachments/SRAM.png)
