## 路由、反向代理

URL 和组件的映射关系，使用 [react router](https://github.com/remix-run/react-router) 实现

css 模块 filename.module.css 可以避免 css 重名，css 中原名字 cssname 成为了 style 中的方法

```jsx
import style from "filename.module.css"
style.cssname
```
