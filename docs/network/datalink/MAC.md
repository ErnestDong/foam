# MAC 地址

MAC 地址是数据链路层的地址，用于唯一标识网络设备。MAC 地址是 48 位的二进制数，通常以 16 进制表示，如 `00:0a:95:9d:68:16`。

通过[[IP]]获得 MAC 地址的过程是 ARP 协议。ARP 协议通过广播的方式，将 IP 地址映射到 MAC 地址。
