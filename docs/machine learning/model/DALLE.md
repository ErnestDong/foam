# DALL E 2

基于[[CLIP]]的扩散模型，有取代[[GAN]]的趋势，有“创造力”。过程是

- prior：给定文本描述生成图像特征
- decoder：从特征生成图像

## 扩散模型

前向扩散过程：给图片加小的噪声 t 次，趋近无穷时趋于噪声
反向生成：模型反向去噪声生成图片，预测噪声有哪些([[CNN&RNN]]里面的ResNet)。模型里面加time embedding，并且共享参数。
