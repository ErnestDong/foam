# 哈希表

最坏情况是 $\Theta(n)$ 但是平均情况 O(1)

散列技术快但是耗费空间大，减少空间占用的方法是利用 [[#hash function]]

## hash function

### 好的散列函数的特点

等可能散列到不同的槽位中

- 除法散列法 $h(k) = k mod m$
- 乘法散列法 $h(k)=m(kA mod 1)$ 向下取整
