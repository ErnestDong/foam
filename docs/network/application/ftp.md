# FTP

可以把本地文件upload到服务器，或download服务器文件到本地。也是基于[[TCP]]的，协议需要支持

- 用户认证：明文传输的
- list 文件
- 上传下载

控制连接和数据连接是分开的，控制连接传输指令，而服务器则反过来与客户端的 22 端口建立数据连接
