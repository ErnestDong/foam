# 二叉树

设 x 是二叉搜索树中的一个结点。如果 y 是左结点，则 x.key>= y.key；反之，x.key <= y.key

## 中序遍历

```python
def inorder_tree_walk(x):
   inorder_tree_walk(x.left)
   print(x.key)
   inorder_tree_walk(x.right)
```

## 查询、插入与删除

这些基本操作都能在 O(h)的时间内完成，h 为二叉树的高度

### 随机构建二叉搜索树

n 个不同关键字的、按随机次序插入构成的二叉树期望高度为 O(log n)

### 平衡二叉树

树的分支较为平衡，方便查找
