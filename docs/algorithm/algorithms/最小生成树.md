# 最小生成树

最小生成树是最小权重，而非最小边数

## Kruskal

是一种[[贪心算法]]，适用于稀疏图。找权重最小的边，并且要保证这条边位于两颗不同的树上

## Prim

与 [[最短路径]] 算法 Dijkstra 算法相似，适合于稠密图，找连接集合 A 的权重最小的边
