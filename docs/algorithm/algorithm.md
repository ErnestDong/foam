## basics

### 数学知识

#### Theta

$f(n)=\Theta(g(n))\leftrightarrow$ f(n)是 g(n)的渐近紧确界，渐进给出了函数的上界和下界#### O

只有渐进上界时采用 O 记

#### Omega

只有渐进下界时采用 Omega 记

### 分治策略

思路是分解、解决、合并，典型例子是 [[归并排序]]通过分解降低问题规模，从而加速算法

### 递归树法

确定递归层数，叶节点为 T(1)

### 主方法

形如 $T(n)=aT(n/b)+f(n)$ 的式子，T(n)有如下渐进界

- $\Theta(n^log_{ba})$
- $\Theta(n^log_{ba}lg n)$
- $\Theta(f(n))$

直觉上是哪个大 T(n)就以哪个为渐进界，如果大小相当要加一个log n
