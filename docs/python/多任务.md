# 多任务

多任务的实现有 3 种方式：

- [[多线程]]：启动一个[[进程]]，在一个进程内启动多个[[并发|线程]]，这样，多个线程也可以一块执行多个任务。
- [[多进程]]：启动多个[[进程]]，每个进程虽然只有一个[[并发|线程]]，但多个进程可以一块执行多个任务
- [[协程]]：我们自己写的程序在逻辑上对每一个任务进行轮询
