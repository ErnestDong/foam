# 财务会计

## 例题

### 税率30% 折扣增加 100 对三张表的影响

1. 利润表：费用增加100，税前利润减少100，所得税费减少30，净利润减少70
2. 资产负债表：固定资产减少100，应交税费应交所得税减少30，未分配利润减少70
3. CF现金流量表：在净利润变动（-70）的基础上调整，加回累计折旧100，当期经营活动产生的现金流增加了30
